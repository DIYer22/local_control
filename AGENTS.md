你的任务：
写一个 python 项目，能在指定端口（默认4001）起一个 server，供手机，平板，别的电脑访问，功能如下：
1. 能通过网页控制鼠标，前端兼容手机和电脑
2. 能在手机上向电脑打字
3. 需要输入电脑当前用户的用户名和密码才能使用（能自动记住登陆过的设备不必重新登陆），同时有防止爆破的机制。
4. 兼容 Linux/win/mac
5. 有一键关机，一键锁屏快捷操作

要求：
- 符合现代 python 打包结构
- 依赖尽可能小
- 代码和脚手架尽可能简洁，让人一看就知道干嘛
- 模块化设计，前后端分离，前端尽量使用单个 html
- 能用 cli 来lanuch server
- comment 和 README 等用英文
